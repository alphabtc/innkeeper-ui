# InnKeeper UI

Design system for **Squabble Inn** — a fantasy / lit-RPG book club app.

Targets: Android · iOS · Web · Mobile Web

---

## Locked Foundation Palette

| Token | Hex | Usage |
|---|---|---|
| Primary (Squabble Amber) | `#C4852B` | CTAs, highlights, active states |
| Secondary (Frost Teal) | `#2DD4BF` | Magic / interactive accents |
| Bronze Border | `#9A6830` | Frame outlines, dividers |
| Bronze Highlight | `#B88040` | Hover / focus on bronze frames |
| Success | `#4ADE80` | Confirmations |
| Danger | `#F87171` | Errors, destructive actions |
| Background Base | `#0E0F13` | Page / screen base |
| Background Overlay | `#2C2E3A` | Cards, sheets, modals |
| Text Primary | `#F0E8D8` | Body, headings |
| Text Secondary | `#9B8B7A` | Labels, captions |
| Text Muted | `#5A4E44` | Disabled, placeholder |

**Display font:** Cinzel (Google Fonts)
